# Sistema de Gestión Logística - Ciudad Guayana
Proyecto de la cátedra de Base de Datos - UNEG.

## Equipo de trabajo
* Roxana Moreno
* Alejandro González
* Juan García

## Tecnologías
* **Base de Datos:** Neo4j (Grafos)
* **Algoritmos:** Dijkstra (Ruta óptima por tráfico)
* **Frontend:** Python + Streamlit + Plotly

## Instrucciones
1. Iniciar la base de datos en Neo4j Desktop.
2. Ejecutar `pip install -r requirements.txt`.
3. Correr la app con `streamlit run app.py`.
